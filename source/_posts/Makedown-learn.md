# 列表
## 无序列表
``` bash
- 列表内容  
+ 列表内容
* 列表内容  
注意：- + * 跟内容之间都要有一个空格  
```
- 列表内容   
	
+ 列表内容
* 列表内容  

## 有序列表
```bash
 1. 列表内容  
 2. 列表内容  
 3. 列表内容   
    注意空格
```

 1. 列表内容  
 2. 列表内容  
 3. 列表内容

# 表格  
``` bash
 表头|表头|表头
---|:--:|---:
内容|内容|内容
内容|内容|内容
第二行分割表头和内容。
-有一个就行，为了对齐,可以多加几个;
文字默认居左
-两边加：表示文字居中;
-右边加：表示文字居右。  
```
